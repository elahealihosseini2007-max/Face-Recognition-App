# Face-Recognition-App

این پروژه، پروژه‌ی پایان‌ترم درس برنامه‌نویسی است که پیاده‌سازی یک سیستم تشخیص چهره تحت وب است.

---

### درباره پروژه (About the Project)
این نرم‌افزار با استفاده از پایتون و فریم ورک Flask ساخته است. تمرکز اصلی این پروژه، درک مفاهیم برنامه‌نویسی شیءگرا (OOP)، پیاده سازی سامانه تشخیص چهره است.


---

### ویژگی‌های کلیدی (Key Features)
*   **تشخیص چهره (Face Detection):** شناسایی حضور چهره در تصویر.
*   **شناسایی هویت (Face Recognition):** تطبیق چهره با دیتابیس موجود.
*   **آپلود تصویر:** امکان بارگذاری عکس توسط کاربر برای پردازش.
